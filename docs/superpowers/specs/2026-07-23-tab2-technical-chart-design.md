# Tab2 技术图表设计

## 目标

在现有 Gradio 交易策略助手中新增 `Tab2 技术图表`。用户可以输入证券代码并自动读取 `outputs/` 中最新的对应 CSV，也可以上传 CSV；系统生成共享时间轴的 Plotly 多面板图，包括 K 线、成交量、BOLL、MACD 和 RSI。

## 架构

- `ui`：负责 Tab2 组件、参数输入和事件绑定。
- `service`：负责 CSV 选择、读取、字段校验、证券过滤、日期过滤、指标计算和图表生成编排。
- `common`：保存请求/结果模型、技术分析异常和字段常量。
- `infra.storage`：扩展 CSV 仓储，提供上传文件读取和按代码查找最新文件。
- `infra.charting`：封装 Plotly 多面板构建，不包含业务判断。

依赖方向保持 `UI -> Service -> Common`，基础设施实现通过构造函数注入 Service。

## 数据输入规则

1. 上传 CSV 优先。
2. 未上传时，按证券代码从 `outputs/` 查找最新 CSV。
3. CSV 有多个证券时，按 `symbol` 或 `instrument_id` 过滤输入代码。
4. 未输入代码且 CSV 只有一个证券时自动识别。
5. 必需字段为 `trade_date, open, high, low, close, volume`。
6. 日期无效、字段缺失、证券不匹配或有效行为空时返回领域错误。

## 指标参数

- MACD：快速 12、慢速 26、信号 9；可在 UI 修改。
- BOLL：周期 20、标准差倍数 2；可在 UI 修改。
- RSI：周期 14；可在 UI 修改。
- 默认显示最近 250 个交易日，允许 50 至 3000。

## 图表

使用一个 Plotly Figure，共五个共享 X 轴的面板：

1. K 线 + MA5、MA10、MA20、MA60、MA250、MA360。
2. BOLL K线及上中下轨。
3. 成交量柱状图。
4. MACD、Signal 和 Histogram。
5. RSI 及 30、70 参考线。

A 股颜色约定：上涨红、下跌绿。禁用范围滑块，启用统一悬停和缩放。
